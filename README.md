# SMART-RAILGUARD-AI-POWERED-DRIVER-DROWSINESS-MONITORING-AND-SAFETY-PLATFORM
The RailGuard AI Driver Drowsiness Detection System is designed to enhance safety in railway transportation by continuously monitoring driver alertness in real time. Driver fatigue is a major cause of accidents, particularly in railway operations where sustained concentration is essential. The lack of an automated monitoring system increases the risk of human error. This project addresses the issue by integrating computer vision, intelligent logic, and real-time analytics to detect drowsiness effectively and prevent accidents.
The system captures live video using a webcam and processes it with OpenCV along with MediaPipe Face Mesh, an advanced facial landmark detection model that identifies 468 facial points. This allows precise tracking of eye and head movements. The system calculates the Eye Aspect Ratio (EAR) to determine whether the eyes are open or closed and monitors blinking patterns over time. It also includes head tilt detection using facial landmark alignment, improving accuracy under different driver positions. Drowsiness is determined using a time-based approach, where prolonged eye closure beyond specific thresholds is classified as DROWSY or CRITICAL. Additionally, a NO DRIVER condition is detected when no face is present in the frame, ensuring complete monitoring.
When drowsiness is detected, the system activates alert mechanisms, including a continuous alarm sound and a voice warning, to immediately regain the driver’s attention. A dynamic safety score is also calculated based on safe and drowsy durations, providing a measurable indicator of driver alertness.
The system is integrated with a Flask-based web application for monitoring and analysis. Driver status, EAR values, and safety scores are logged into a SQLite database. A user-friendly dashboard displays real-time updates, event logs, and visual analytics such as trend graphs and status distribution. Additionally, the system includes performance evaluation metrics like confusion matrix, accuracy, precision, recall, and F1-score.
Overall, RailGuard AI is a cost-effective, scalable, and non-invasive solution that demonstrates the effective use of artificial intelligence in improving transportation safety and reducing fatigue-related accidents.
